async databaseFiles.initializeDatabase.create_schema()[source]¶

Creates and initializes the database schema for a hotel reservation system.

This function creates several collections in the MongoDB database, inserts initial data, creates indexes, and purges dummy data. The collections created include room types, rooms, customers, reservations, and room bookings.
