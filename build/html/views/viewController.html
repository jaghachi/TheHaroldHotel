views/viewController module¶

” class: ViewController most recent update: 8/7/2024 programmers: Jonathan Aghachi

Description: The ViewController class manages the different views in the application using a QStackedWidget. It allows for easy navigation between views, adding and removing views as needed, and maintaining the overall structure of the application’s user interface.

class views.viewController.ViewController[source]¶

Bases: QStackedWidget

This class provides methods to add, show, remove, clear, and re-add views within the stacked widget. It’s central to the application’s navigation flow, ensuring that the correct view is displayed based on user interactions.

None specific to the instance; inherits from QStackedWidget.
